# Learning by List

Turn Anki into a faster desktop review workspace by browsing the current deck as a note-based list.

This add-on is built to improve study efficiency when normal card-by-card review feels too narrow. It makes Anki better suited for repeated review, quick scanning, and cross-checking multiple fields at once.

## What It Does

- Adds a `Learning by List` button to the deck overview screen.
- Opens the current deck in a separate list view.
- Lets you choose which fields are visible.
- Remembers visible columns, page size, and filter state per deck.
- Renders audio fields as compact play buttons when `[sound:...]` tags are present.

## Good Fit For

- Learners who want a faster high-volume review workflow.
- Large decks where you want to inspect many notes quickly.
- Vocabulary decks with multiple text fields.
- Audio-heavy notes where you want to preview pronunciation inline.
- Repeated review sessions where you want Anki to feel more like a study workspace than a strictly one-card-at-a-time interface.

## Compatibility

- Designed for desktop Anki only.
- Validated locally on Anki `25.02.7`.
- If you try it on other desktop versions, compatibility reports are very welcome.
