# Changelog

## 0.1.0

- Initial public release.
- Adds a `Learning by List` button to the deck overview screen.
- Opens the current deck in a note-based list window.
- Supports configurable columns, page size, and per-deck persisted view state.
- Renders audio fields as compact play buttons when audio tags are present.
- Includes a local packaging script for clean `.ankiaddon` builds.
