# Learning by List Config

`page_size_default`

- Default number of note rows shown per page.

`page_size_max`

- Maximum page size allowed in the UI.

`default_visible_column_count`

- How many note fields are shown by default when a deck is opened for the first time.

`audio_field_name_hints`

- Used only as UI hints.
- If a field name contains one of these strings, it is more likely to be treated as an audio-oriented column label.
- Actual audio buttons are rendered based on `[sound:...]` tags in the field content.

`system_columns`

- Extra non-note-field columns that can be shown in the list view.

`per_deck`

- Stores per-deck UI state such as visible columns and page size.
